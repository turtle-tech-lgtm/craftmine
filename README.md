# 🐉 Craftmine

A 2D side-scrolling Minecraft-inspired adventure built with Phaser 3. Traverse dangerous dimensions, collect Minecoins, and prepare for the ultimate showdown in The End!

## 🎮 How to Play
1. **Enter Your Name**: Start by entering your name on the main menu to track your speedrun time.
2. **Collect Coins**: You need to collect **20 Minecoins** in each level to progress to the next.
3. **Survive**: Fight off Zombies, Skeletons, and exploding Creepers.
4. **The End**: Reach Level 6 to face the **Ender Dragon**. Defeat it to win the game and record your time on the leaderboard!

## ⌨️ Controls
| Action | Key |
| :--- | :--- |
| **Move Left / Right** | `A` / `D` |
| **Jump** | `W` / `Space` |
| **Attack (Sword)** | `J` |
| **Shoot (Bow)** | `K` |
| **Block (Shield)** | `L` |
| **Mine (Pickaxe)** | `U` |
| **Build (Place Block)** | `I` |
| **Pause** | `ESC` |

## 🛠️ Features
- **Procedural Levels**: Every level is randomly generated for a unique experience every time.
- **Combat System**: Use your sword for melee, your bow for ranged attacks (vertical in the boss fight!), and your shield to block damage.
- **Mining & Building**: Use your pickaxe to clear terrain or place blocks to build bridges and safety walls.
- **Boss Fight**: A fully realized Ender Dragon battle with a health bar and custom mechanics.
- **Leaderboard**: Compete for the fastest total time!
